# NumPy Learning Notes

This repository contains my structured learning notes and experiments while studying **NumPy**, the fundamental library for numerical computing in Python.

The notebooks focus on understanding how NumPy arrays work, how they differ from Python lists, and how to efficiently perform numerical operations using vectorized computations.

---

## Topics Covered

* NumPy arrays and basic concepts
* Creating arrays from lists and from scratch
* Differences between Python lists and NumPy arrays
* Vectors, matrices, and higher-dimensional arrays
* Array properties (`dtype`, `shape`, `size`, `ndim`)
* Array transformations (`reshape`, `flatten`, `ravel`, `transpose`)
* Indexing and slicing
* Sorting arrays
* Boolean filtering and masking
* Fancy indexing and `np.where()`
* Adding and removing data (`concatenate`, `vstack`, `hstack`, `delete`)
* Shape comparison
* Performance comparison between Python lists and NumPy operations
